# Скриншоты для дипломной работы

В этой папке нужно сохранить реальные скриншоты интерфейса перед финальной сборкой Word-файла.

Рекомендуемые имена файлов:

- `figure_3_1_home_page.png` — главная страница сервиса.
- `figure_3_2_registration.png` — форма регистрации.
- `figure_3_3_chat.png` — личный чат пользователя.
- `figure_3_4_specialists_map.png` — каталог специалистов и карта.
- `figure_3_5_telegram_start.png` — Telegram-бот после команды `/start`.
- `figure_3_6_telegram_emergency.png` — Telegram-бот с командой `/emergency`.
- `figure_3_7_admin_model_versions.png` — управление версиями модели в Django Admin.
- `figure_3_8_admin_safety_audit.png` — журнал safety-аудита.
- `figure_3_9_admin_emergency_resources.png` — управление экстренными ресурсами.

В тексте `08_diploma_v2_extended.md` уже добавлены ссылки и подписи к этим рисункам. После добавления реальных изображений их можно вставить в Word на отмеченные места.
